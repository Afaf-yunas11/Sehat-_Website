# sehat
An application designed to connect patients with healthcare professionals

## Features

- **User Authentication**: Secure login and registration for patients and healthcare professionals.
- **Appointment Scheduling**: Patients can book appointments with available healthcare professionals.
- **Profile Management**: Users can manage their personal and professional profiles.
- **Search Functionality**: Patients can search for healthcare professionals by specialization, location, or availability.

## Installation

1. Clone the repository:
  ```bash
  git clone https://github.com/yourusername/sehat.git
  ```
2. Navigate to the project directory:
  ```bash
  cd sehat
  ```
3. Install dependencies:
  ```bash
  npm install
  ```
4. Set up the environment variables:
  - Create a `.env` file in the root directory.
  - Check the [.env.example](.env.example) and setup environment variables accordingly
5. Start the application:
  ```bash
  npm start
  ```

## Usage

1. Open your browser and navigate to `http://localhost:your_port_number`.
2. Register as a patient or healthcare professional.
3. Log in to access the dashboard.
4. Use the search functionality to find healthcare professionals or manage your appointments.

## Technologies Used

- **Frontend**: React
- **Backend**: Node.js, Express.js
- **Database**: Microsoft SQL Server
- **Authentication**: JSON Web Tokens (JWT)
